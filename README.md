# Dental Clinic Call Analytics Dashboard

A Streamlit-based dashboard to monitor and analyze call activities for a dental clinic.  
This dashboard provides key metrics, visual insights, AI-powered call classification, sentiment analysis, call quality scoring, and a booking funnel.

---

# Requirements

- Streamlit
- pandas
- plotly
- nltk
- openpyxl

---

## Setup & Deployment Instructions

1. **Push your project to GitHub**  
   - Ensure your repository contains all necessary files:
     - `app.py` (Streamlit script)
     - `Assignment Dataset.xlsx`
     - `requirements.txt` (listing all dependencies)

2. **Deploy using Streamlit Cloud**  
   - Go to [Streamlit Community Cloud](https://share.streamlit.io/) and log in.  
   - Click **“New app”**.  
   - Connect your GitHub account and select your repository.  
   - Choose the branch and main file (e.g., `app.py`).  
   - Click **Deploy**. Streamlit will automatically install dependencies and host your app online.

3. **Access your app**  
   - After deployment, you will get a public URL to access your dashboard.  
   - Any future updates pushed to GitHub will automatically update the live app on Streamlit Cloud.
